# Line Edit Checklist

Use this reference for `逐字分析`, sentence-level polishing, paragraph tightening, and micro-continuity review.

## Output Format

```text
Original:
Function:
Problem words/details:
Issue type:
Why it matters:
Suggested edit:
Permission level:
Continuity risk:
Style risk:
```

## Sentence-Level Checks

1. Identify the sentence function: action, reaction, exposition, transition, atmosphere, clue, voice, rhythm, or payoff.
2. Check whether every word contributes to function, voice, rhythm, or continuity.
3. Check subject clarity and pronoun reference.
4. Check action order: perception, reaction, decision, movement, consequence.
5. Check whether physical action is possible in the scene space.
6. Check whether emotional response follows a visible trigger.
7. Check whether information belongs in this point of view.
8. Check whether the sentence reveals future information too early.
9. Check whether a detail is foreshadowing before cutting it.
10. Preserve intentional roughness, dialect, silence, and compression when they serve voice.

## Word-Level Checks

| Target | Ask |
|---|---|
| Verb | Is it precise enough, or falsely dramatic? |
| Adjective | Does it create image, judgment, tone, or redundancy? |
| Adverb | Does it add behavior not already in the verb? |
| Emotion word | Should this be embodied, hidden, or left direct? |
| Modifier stack | Does it slow the sentence without adding control? |
| Repeated word | Is repetition rhythmic/intentional or accidental? |
| Filter phrase | Does "saw/felt/heard/realized" help POV or add distance? |
| Time marker | Does the order of events remain clear? |

## Micro-Edit Priorities

Prefer edits in this order:

1. Remove contradiction or ambiguity.
2. Restore action and emotional order.
3. Clarify viewpoint and referents.
4. Cut accidental repetition.
5. Sharpen verbs and images.
6. Adjust rhythm.
7. Preserve voice.

Do not make every sentence ornate. Strong prose includes plain sentences, silence, and functional transitions.
