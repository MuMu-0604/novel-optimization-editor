# Final QA Checklist

Use this reference before presenting an optimization as complete, after any authorized rewrite, and before whole-book reports.

## Scope Verification

```text
Requested scope:
Actual processed scope:
Coverage manifest status:
Partial or complete:
Unprocessed gaps:
Claims limited to processed material: yes / no
```

## Continuity QA

Check:

```text
Plot facts changed without authorization:
Timeline changed:
Narrative order changed:
Reveal order changed:
Character knowledge changed:
Relationship state changed:
World rule changed:
Foreshadowing setup/payoff affected:
Open question created:
Open question accidentally closed:
```

## Edit QA

Check:

```text
Problem solved:
Smallest sufficient edit used:
Permission level respected:
Risk level logged:
Author confirmations obtained:
Style target preserved:
Dialogue voices preserved:
Records synced:
Rollback notes available:
```

## Whole-Book QA

Before giving whole-book conclusions, verify:

1. Coverage manifest is complete.
2. Chapter Index covers all known chapters.
3. Timeline has no unexplained event-order contradiction.
4. Reveal Order has no accidental early reveal.
5. Character State Table supports major decisions and emotional turns.
6. Foreshadowing Ledger marks setup and payoff status.
7. Open Questions are either intentional, pending, or flagged.
8. Recommendations distinguish evidence-based fixes from optional inventions.

## Final Response Format

```text
Completed scope:
Main improvements:
Unresolved risks:
Records updated:
Author decisions still needed:
```
